# Changelog

All notable changes to NeoCheck will be documented in this file.

## [1.0.0] - 2026-07-02
### Added
- **Core Engine:** Concurrent pipeline execution of IP inspection providers.
- **Diagnostics Dashboard:** Premium Apple/Linear-inspired client diagnostic landing page with framer-motion.
- **Admin panel:** shadcn/ui inspired settings dashboard.
- **Stateless Persistence:** Complete mapping to `/opt/neocheck` via volumes.
- **Installer:** Interactive `install.sh` script for automated Ubuntu installation.
- **Security:** Standard JWT HTTP-only sessions, secure response headers, and client rate limits.
- **CI/CD:** GitHub Actions workflows for compilation validation, Docker builds, and releases.
