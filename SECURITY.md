# Security Policy

## Reporting a Vulnerability

We take the security of NeoCheck seriously. If you find any vulnerabilities or security issues in this project, please do **NOT** open a public issue. 

Instead, report it directly to the maintainers:
- **Email:** security@example.com (Please replace with actual email)

Please include:
- A descriptive summary of the issue.
- Detailed steps to reproduce the vulnerability (proof of concept).
- Potential impact of the issue.

We will acknowledge receipt of your report within 48 hours and work with you to patch the issue before making a public announcement.
